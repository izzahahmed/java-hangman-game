# Hangman Game (Java)

This is a console-based Hangman game written in Java.

## Description
The program randomly selects a word and allows the user to guess letters with a limited number of attempts. The game displays progress after each guess and ends when the word is guessed or attempts run out.

## Features
- Random word selection
- Limited guesses
- Input validation
- ASCII-based hangman display

## Technologies Used
- Java
- Scanner for user input
